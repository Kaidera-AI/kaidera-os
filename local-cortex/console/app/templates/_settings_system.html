System
Console configuration for the Cortex connection, external harnesses, and app preferences. Stored locally in the console app-DB and applied on reload. This is the console's own store, not the real Cortex .env.
Secrets live only in the local console settings store — never committed, never rendered back into the page. A stored key shows as •••• set; use Replace to enter a new one. Local single-user store.
